#  ユーザーの心のケアをするためのaiチャットアプリ
#このアプリの概要
·最近のaiは数学的賢さやより人に寄り添う形のaiが増えてきましたそこでまだ人の心のケアする為のaiはまだ少ないですだから僕はそれを叶える為にこのアプリを考えました。
多種多様な環境を想定してどの端末でも使えるようにします。
<img width="500" height="879" alt="image" src="https://github.com/user-attachments/assets/2693d433-85a7-4f1d-8ad4-675a58100537" />


まずユーザーのリクエストをナレッジでなりきってほしいキャラクターのペルソナになります。
リクエストには音声認識のwhisperをMCPと呼ばれるai自ら呼び出せるようになったapiでaiがこの言葉はノイズじゃなくて話していると判断したときに次のフローに進みます。
そして第二のナレッジでユーザーに対しての心の救済を考えます。
aiにはプライバシーが重要なのでollamaと呼ばれるランタイムを使用してwifitがない環境でも使える誰にも漏れない安全なチャットになっています。
チャットだけでは心が動かないことも考慮してlive2dによるキャラクターの笑顔で励ましてもらえます。
