アプリの概要】
近年のAIは数学的・論理的な賢さが目立つ一方、人間の心に真に寄り添い、メンタルケアを施すプロダクトは未だ少ないのが現状です。本プロジェクトは、多種多様な人やデバイス環境を想定し、誰もが安心していつでも「心の救済」を得られる世界を目指して設計したAIチャットアプリケーションです。

【技術的特徴とアーキテクチャのこだわり】

・2段階のナレッジコンテキスト（Pipeline）：
コンテキストの入力層を２つに離しています。第１で「なりきってほしいキャラクターのペルソナ（キャラクター特性）」を反映し、第２で「心のケアに関する専門知識（メンタルケアの専門知識）」をナレッジとしてAIに注入します。これにより、キャラクター性を維持したまま、専門的で安全なケアメッセージを生成する仕組みを担保しています。

・Ollamaによる完全ローカル推論とプライバシー保護：
メンタルケアの相談内容は極めてセンシティブな個人情報です。そのため、本システムはクラウドAPIに依存せず、ローカルLLMランタイムである「Ollama」を採用しました。完全なオフライン環境（Wi-Fiがない環境）でも動作し外部サーバーに一切データを漏洩させない、究極のプライバシーと安全性を両立したチャット環境を実現しています。

今現在チャット画面からキャラクターを呼び悩みを聞いてもらうところまではできました。これからは以下の機能を実装していきたいと考えています。

・ MCP（Model Context Protocol）によるエコシステム構築：
ユーザーの音声リクエストは、MCPを介して音声認識モデルである「Whisper」を動的に呼び出す設計にしています。音声入力からノイズを除去し、人間が話していると判定されたセグメントのみを正確に抽出して次フローへ受け渡します。出力側でもMCPサーバーを介して「Live2D」を制御し、会話内容に同期したキャラクターの笑顔やモーションでユーザーを視覚的に励まします。
